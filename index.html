You are viewing the setup page of Daedalus-SNAppCrafter

Your webpages will replace the current index.html file in the /home/$USER/snapp directory.
Simply stop the snapp server through stopping nginx service 'sudo systemctl sto nginx'.
Replace the files in the snapp directory to host your own SNApp. To restart the SNApp server
enter 'sudo systemclt start nginx' and the server will restart again.
You will likely need to clear your browser cache to see the new SNApp.
